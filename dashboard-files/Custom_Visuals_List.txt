## Installation Instructions

### Method 1: Power BI Desktop AppSource
1. Open Power BI Desktop
2. Go to Home â†’ Get Data â†’ More â†’ AppSource
3. Search for visual name or AppSource ID
4. Click "Add" to install
5. Visual appears in Visualizations pane

### Method 2: Power BI Service (Online)
1. Login to powerbi.microsoft.com
2. Go to Settings â†’ Custom visuals
3. Click "Add a custom visual"
4. Select "From AppSource"  
5. Search and install required visuals

### Method 3: Import from File (.pbiviz)
1. Download .pbiviz file from vendor
2. Power BI Desktop â†’ Import visual from file
3. Select downloaded .pbiviz file
4. Visual appears in Visualizations pane

## MCCNO Brand Configuration

### Visual Color Themes
Apply MCCNO brand colors to each custom visual:

**Primary Colors:**
- Navy (#003366) - Main data series, headers
- Golden (#FFB500) - Highlights, positive indicators
- White (#FFFFFF) - Backgrounds, text on dark
- Neutral (#F5F5F0) - Secondary backgrounds

**Status Colors:**
- Success: #28A745 (Green)
- Warning: #FFC107 (Amber) 
- Danger: #DC3545 (Red)
- Info: #007BFF (Blue)

### Typography Settings
- Font Family: Segoe UI
- Header: 18pt Bold Navy
- Body: 14pt Regular Navy
- Caption: 12pt Regular Gray

## Support & Troubleshooting

### Common Issues
1. **Visual not appearing**: Check Power BI Desktop version compatibility
2. **Brand colors not applying**: Verify JSON theme import
3. **Performance issues**: Reduce data model complexity
4. **Mobile display problems**: Test custom visual mobile support

### Vendor Support
- **OKViz Visuals**: support@okviz.com
- **Microsoft Visuals**: Built-in support through Power BI
- **Third-party Visuals**: Check individual vendor documentation

### Alternative Solutions
If custom visuals unavailable:
- Use built-in visuals with enhanced formatting
- Apply conditional formatting for brand colors
- Create calculated columns for enhanced functionality
- Utilize bookmarks for interactive navigation
